### What is a Sentinel-X ###
Sentinel-x is actually something I created for my own convenience. I do my coding work at night, and since I'm into the cyber scene (hacker vibe), there is a hidden under hacker todo inside this. This is a 'double todo' app I built to manage daily tasks—outwardly, it's for adding normal everyday activities, but inwardly, it's for doing tech-related secret operational todos like coding.

---
### Simple Features💡 ###
* No paper needed to write down easily forgotten tasks.
* helps avoid task-related stress.
* simple loading and simple working.
* Type root--access to experience the cyber hacker todo mode.🧑‍💻
* A shower of flowers when a task is completed.🎉
* Data is saved safely on the web using LocalStorage API, so it won't disappear.

### MY Stress up times ###
